# REST framework

##  Install
```
pip install -r requirements.txt
```

## Usage

### Makemigrations 

```shell
python manage.py makemigrations
python manage.py migrate
```

### Create Superuser
```shell
python manage.py createsuperuser
```

### Restore
```shell
./restore.sh
```

### Start Django
```shell
python manage.py runserver 127.0.0.1:8000
```
